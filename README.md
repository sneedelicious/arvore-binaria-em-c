# Estrutura de Dados: Árvore Binária de Busca (BST)

Este projeto foi desenvolvido como parte da disciplina de **Estrutura de Dados** na faculdade. Ele implementa uma Árvore Binária de Busca em linguagem C, cobrindo as operações fundamentais de manipulação de nós.

## 🚀 Funcionalidades
- **Inserção:** Adiciona valores mantendo a propriedade da BST.
- **Remoção:** Trata os três casos clássicos (nó folha, um filho ou dois filhos).
- **Percursos (Traversals):** - Pré-ordem
  - Em ordem (exibe os dados ordenados)
  - Pós-ordem
- **Visualização:** Impressão gráfica da estrutura no console.
- **Gestão de Memória:** Liberação completa da árvore ao encerrar.

## 🛠️ Como Executar
Para compilar e rodar o programa, você precisará de um compilador C (DEV C++).
